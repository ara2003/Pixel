# Pixel

## Опсание

Мессенджер из двух микросервисов: авторизации и ресурсов.

## Технологии

React, TypeScript - для клиентской части.

Spring, Java - для сервера.

OAuth2 - для авторизации.

WebSocket - для push-уведомлений клиента.
